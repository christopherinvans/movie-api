MyMovieApp

Welcome to MyMovieApp, a project that utilizes a REST API to interact with a database of movie info. Enjoy!

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies in the database
Get data about a single movie by name /movies/[Name] GET None A JSON object holding data about a single movie, containing description, year, genre, director, image URL, whether movie is featured or not. Example:
{
name: "The Cat in the Hat",
year: 2003,
genre: "Comedy",
director: "Bo Welch",
image URL: "sampleurl",
featured: no
}
Get data about a genre by name/title /genre/[Name] GET None A text message describing the genre of film
Get data about a director by name /directors/[Name] GET None A JSON object holding data about a single director by name, including bio, birth year and death year. Example:
{
name: "Bo Welch",
bio: "insert short blurb here",
birth year: 1951,
death year: TBD
}
Register new user /users/[Name] POST None A text message indicating whether the user was successfully added
Allow users to update user info (username) /users/[Name] PUT None A text message indicating whether username change was successfully changed
Allow users to add movie to favorites list /movies/myfavorites/[Name] POST A JSON object holding data about the movie to add. Example:
{
name: "The Cat in the Hat",
year: 2003,
genre: "Comedy",
director: "Bo Welch",
image URL: "sampleurl",
featured: no
} 		A JSON object holding data about the movie that was added. Example:
{
name: "The Cat in the Hat",
year: 2003,
genre: "Comedy",
director: "Bo Welch",
image URL: "sampleurl",
featured: no
}
Allow users to remove movie from favorites /movies/myfavorites/[Name] DELETE None A text message indicating whether movie was removed from favorites list
Allow existing users to derigister /users/[Name] DELETE None A text message indicating whether user was successfully removed
