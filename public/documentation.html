MyMovieApp

Welcome to MyMovieApp, a project that utilizes a REST API to interact with a database of movie info. Enjoy!

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies in the database
Get data about a single movie by name /movies/[Name] GET None A JSON object holding data about a single movie. Example:
Title: "The Cat in the Hat",
Description: "Mike Meyers is unhinged and it is glorious.",
Genre: {
Name: "Comedy",
Description: "Giggle fodder"
Director: {
Name: "Bo Welch",
Bio: "He made movies and stuff"
}
Actors: "Mike Meyers and others",
Featured: N
}
Get data about a genre by name/title /genre/[Name] GET None A JSON object containing Genre info.
Get data about a director by name /directors/[Name] GET None A JSON object holding data about a single director by name.
Register new user /users/[Name] POST None A JSON object containing the new user's data, formatted as follows:
{
Username: example123
Password: hunter2
Email: name@name.com
Birthday: 2001-01-14
}
Allow users to update user info (username) /users/[Name] PUT None A JSON object containing the new user's data, formatted as follows:
{
Username: example123
Password: hunter2
Email: name@name.com
Birthday: 2001-01-14
}
Allow users to add movie to favorites list /movies/myfavorites/[Name] POST A JSON object holding an array of the user's favorite movies. A JSON object holding data about the movie that was added. }
Allow users to remove movie from favorites /movies/myfavorites/[Name] DELETE None A text message indicating whether movie was removed from favorites list
Allow existing users to derigister /users/[Name] DELETE None A text message indicating whether user was successfully removed
